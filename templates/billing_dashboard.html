📚 Panduan Penggunaan Dashboard
📶 Active Connection itu apa?

Tab ini menampilkan pelanggan yang sedang online real-time. Gunakan untuk pantau siapa yang sedang terhubung sekarang.

Fungsi utama:

  • Name: nama user PPP yang sedang aktif.
  • Address: IP user saat terkoneksi.
  • Uptime: lama user online (format singkat).
  • Aktif Mulai: estimasi waktu mulai koneksi.

Jika perlu putus koneksi dari sistem, klik tombol 🔌 Disconnect di baris user tersebut.

🔒 PPP Secret itu apa?

PPP Secret adalah akun pelanggan (username dan password) yang dipakai pelanggan untuk login internet di router Anda.

Aksi tombol yang tersedia:

  • ➕ Tambah Secret untuk menambahkan akun pelanggan baru.
  • ✎️ Edit untuk ubah password, profile, local address, atau remote address.
  • 🚫 Disable / ✅ Enable untuk menonaktifkan atau mengaktifkan akun tanpa menghapus.
  • 🗑️ Hapus untuk menghapus akun permanen.

Catatan: saat ✎️ Edit, kolom username dibuat readonly supaya identitas akun tidak berubah.

🧩 PPP Profile itu apa?

PPP Profile berisi aturan paket internet, misalnya rate-limit, batas satu login, dan parameter lain.

Di dashboard ini tab Profile masih untuk pemantauan (read-only). Pengaturan detail profile sementara tetap dilakukan di router pemilik agar konfigurasi utama tetap aman.

💳 Pembayaran dipakai untuk apa?

Tab ini dipakai untuk mencatat pembayaran pelanggan per bulan dan memantau yang sudah bayar atau yang masih menunggak.

Aksi penting:

  • 💳 Bayar: pilih pembayaran sampai bulan tertentu.
  • ❌ Batal Bayar: kembalikan status jika terjadi salah klik.
  • 💰 Input Paket: isi atau ubah nominal bulanan pelanggan.
  • 🖨️ Print Daftar Pembayaran: cetak form penagihan lapangan (A4).

Jika tagihan router belum lunas, hanya menu tertentu yang bisa diakses sampai admin konfirmasi pembayaran router.

📜 Log itu apa?

Tab Log adalah riwayat aktivitas penting agar mudah audit bila ada komplain atau salah input.

Yang tercatat misalnya: aksi bayar, batal bayar, ubah nominal paket, dan waktu kejadian.
